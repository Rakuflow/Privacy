## [0.1.0] - 2026-02-27

### Added
- Initial release of the RakuFlow project on Starknet.
- Privacy-first transfer model based on a shielded pool approach.
- End-to-end user flow: wallet connect, shielded identity setup, deposit, private transfer, and withdraw.
- Core project components:
  - Frontend dApp
  - Cairo contract package
  - Zero-knowledge circuit package
  - Integration and operational scripts
- Release governance baseline:
  - Versioned release artifacts
  - Branching strategy and release process documentation
